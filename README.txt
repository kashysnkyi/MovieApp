# MovieApp

ASP.NET Core MVC application connected to PostgreSQL database.

## Project description
This project was created for laboratory work stages 1.2 and 1.3.

## Implemented
- Connection to existing PostgreSQL database
- Database First approach with Entity Framework Core
- Models generated from database
- Controllers and Views for:
  - Films
  - Genres
  - Users
  - Comments
  - Ratings

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL
- pgAdmin 4
